Manual Deployment
=================

This section walks you through the manual deployment of a Debian system using a PXE server and a
preseed file. This is a great way to learn the ins and outs of the process.

By following the steps in this section, you will learn to configure a Debian VM to act as a PXE
server, and use it to deploy a Debian system on a target machine. You will also learn how to
automate the Debian installation process using a preseed file.
